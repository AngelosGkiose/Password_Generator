# Password Generator 

A simple command-line password generator built with Python.

The program generates a secure random password containing letters, numbers, and special characters. It also displays how many letters, numbers, and symbols the generated password contains.

## Features

- Custom password length
- Input validation
- Random password generation
- Includes:
  - Letters
  - Numbers
  - Symbols
- Character statistics

## Concepts Practiced

- Functions
- Lists
- Loops
- Input validation
- Exception handling (try / except)
- Python random module
